# ⚽ Penaltis por la Seguridad

Juego interactivo de penaltis para reforzar la cultura de seguridad en Alegra.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy a GitHub Pages

```bash
$env:GITHUB_PAGES="true"; npm run build
```

Luego sube la carpeta `out/` a GitHub Pages desde `Settings > Pages > Deploy from a branch`.
